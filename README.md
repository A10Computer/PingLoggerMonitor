Ping-Logger-Monitor a lightweight network monitoring tool - 
Features:

🔘 One-click ping for predefined targets (Google, Yahoo, Bing)

🌐 Custom hostname/IP input

⏱ Auto-refresh every 4 seconds

📊 Large real-time latency display (ms)

🎨 Color-coded status (green,yellow,red)

💾 Optional logging to file

🛑 Start / Stop control

🚀 Quick access to Speedtest.net

The .zip Contains the file PingMonitor.exe. No administrator rights required.
